# 🚀 AI-Powered Resume Screener 2026

An intelligent resume analyzer that scores your resume against any job description using Groq + Llama-3.3-70B.

**Features:**
- Upload PDF resume
- Paste job description
- Get Match Score (0-100)
- Strengths, Weaknesses, Improvements & ATS tips

**Tech Stack:** Streamlit, PyMuPDF, Groq LLM
